# Whisker Wellness V2

A web application to help cat owners track their cat's weight and manage weight loss.

## Features

- User registration and login
- Add and manage cat profiles
- Log daily weights for cats
- View weight graphs for the last 7 days

## Installation

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm start` to start the server
4. Open http://localhost:3000 in your browser

## Usage

1. Sign up for an account or log in
2. Add your cats with their details
3. Log weights daily for each cat
4. View graphs to track progress

## Acceptance Criteria

- AC-1.1: Given home page, when sign in, then show cat profiles
- AC-1.2: Given weight info, when make a graph, then make an accurate graph for the week

## Database

Uses SQLite database with tables for users, cats, and weight logs.
